# Beau Tea Operations Optimization 
## Project Overview
Business analytics project for a DC-based beverage startup, developing 4 optimization models to improve staffing, marketing allocation, and machine throughput. Analyzed 1,000+ demand scenarios using Integer Linear Programming, Monte Carlo Simulation, Risk Modeling, and Decision Trees.

## Key Results

| Area | Optimization |
|------|--------------|
| Staffing Costs | **25% reduction** |
| Marketing Allocation | **10% improvement** |
| Machine Throughput | **30% increase** |
| Scenarios Simulated | **1,000+** |
| Queuing Simulations | **480** |

## Tech Stack
- **Languages:** Python, R
- **Libraries:** PuLP, SimPy, NumPy, Pandas, Scikit-learn
- **Techniques:** 
  - Integer Linear Programming (ILP)
  - Monte Carlo Simulation
  - Risk Modeling
  - Decision Trees
  - Queuing Theory
